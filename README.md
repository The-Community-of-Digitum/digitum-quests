# 🧩 Digitum Quests

This repository contains the official quest system of **The Republic of Digitum**.

Quests are the primary way citizens earn D$.  
All quests must follow the rules and processes defined here.

Digitum uses a **two‑PR verification system**:

1. **Doer PR** — the citizen who completed the quest submits a request to earn D$.
2. **Witness PR** — a separate citizen confirms the completion.

Only after both PRs are approved will the reward be added to the ledger.
