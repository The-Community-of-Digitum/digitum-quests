# 👁️ Witness Verification Process (PR #2)

A witness must be a Tier 1 Citizen or higher.

To verify a quest:

1. Review the doer's PR.
2. Confirm the proof is valid.
3. Create a second PR with a file:
   `/logs/witness-<questname>-<witness>-<date>.md`
4. Include:
   - confirmation statement
   - link to the doer PR
   - any additional notes

Example:

```
Witness Verification — "Fix the Breakfast Sign"
Witness: craftQueen
Doer PR: #42
Verification:

Proof is valid

Work matches quest requirements
Approved: Yes
```

Only after this PR is approved may the reward be added to the ledger.
