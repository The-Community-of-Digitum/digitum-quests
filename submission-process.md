# 🧑‍🔧 Doer Submission Process (PR #1)

To claim a quest reward, the doer must:

1. Complete the quest.
2. Create a Pull Request in this repo.
3. Add a file under `/logs/` named:
   `doer-<questname>-<username>-<date>.md`
4. Include:
   - proof of completion
   - link to the quest
   - explanation of how requirements were met

Example:

```
Quest Fulfillment — "Fix the Breakfast Sign"
Doer: waffleMaster
Quest: /quest-board/fix-breakfast-sign.md
Proof:

Photo of repaired sign

Commit link to design file
Notes:
Completed on 2026-06-07
```

This PR must be approved before the witness PR can proceed.
