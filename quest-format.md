# 📝 Quest Format

All quests must follow this structure:

```
Title: <Quest Name>
Reward: <Amount of D$>
Category: <Category Name>
Difficulty: <Easy / Medium / Hard>
Description:
<What the quest requires>
Requirements:

<Requirement 1>

<Requirement 2>
Proof Needed:

<What the doer must show>
Witness Requirements:

<What the witness must verify>
```

Quests must be fair, creative, and non‑exploitative.

For the witness' side, they must create a PR with those witness requirements fulfilled and a required stamp:
`I Approve of this Quest in Digitum`.
